# 【完全図解】素人でもできるWebアプリ無料公開＆広告収益化ガイド

しんたろうさんが今回作成した「Auto-Prompt Builder PRO」を、**ドメイン代やサーバー代ゼロで世界中に公開**し、さらに**Google AdSenseを使って自動収入を得る**ための3ステップを解説します。
専門的なプログラミング知識は不要です。マウス操作だけで完了します。

---

## 🚀 ステップ1：サイトをインターネットに無料公開する（Netlify Drop）

Webサイトを公開するためのサービスはいくつもありますが、初心者にとって一番直感的で簡単なのが**「Netlify（ネットリファイ）」**というサービスです。

### 全くのアカウントなしから公開する手順
1. パソコンのブラウザで **[Netlify Drop](https://app.netlify.com/drop)** にアクセスします。
2. 画面に大きな円（点線で囲まれたアップロードエリア）が表示されます。
3. しんたろうさんのパソコンの中にある、**`01_Auto_Prompt_Builder` フォルダの中の `public` という名前のフォルダを**、その円の中にドラッグ＆ドロップします。
   - ※`index.html`ファイル単体ではなく、必ず**「publicフォルダごと」**落としてください。（これにより、このマニュアル等の不要なファイルがネット上に一緒に公開されるのを防げます）
4. 数秒待つと、画面が切り替わり、緑色の文字で「**Published**」と表示されます。
5. その画面にランダムなアルファベットのURL（例： `https://happy-goldberg-12345.netlify.app`）が表示されます。これが**あなたの新しいWebアプリのURL**です！
6. **【重要】**サイトを消されないように、右上のボタンから「Sign up（無料会員登録）」を行ってください（Googleアカウントなどで1クリックで登録可能です）。

これで公開は完了です！このURLをX（Twitter）などでシェアすれば、誰でもツールを使えるようになります。

---

## 💰 ステップ2：Google AdSense（広告）に申し込む

サイトにアクセスが集まった際に、クリックや表示だけで報酬が発生する「Google AdSense」を導入します。

1. **[Google AdSense の公式サイト](https://adsense.google.com/intl/ja_jp/start/)** にアクセスし、「ご利用開始」をクリックします。
2. お持ちのGoogle（Gmail）アカウントでログインします。
3. 登録画面にて、**「ステップ1」でNetlifyから発行されたURL**を「お客様のサイト」に入力します。
4. 画面の指示に従い、お客様情報や支払いを受け取るための情報を入力します。
5. 最後に「サイトをAdSenseにリンク」という手順で、**AdSense側の審査**が開始されます。
   - ※審査には数日〜数週間かかる場合があります。

---

## 🔧 ステップ3：広告タグを貼り付ける（審査通過後）

AdSenseの審査を無事通過すると、管理画面から「**広告ユニット（広告タグ）**」という短いコード（プログラムの切れ端）が発行できるようになります。

1. AdSense管理画面で「広告」＞「広告ユニットごと」＞「ディスプレイ広告」を選択して作成ボタンを押します。
2. `<script async src="...` で始まる数行のコード（広告タグ）が表示されるのでコピーします。
3. しんたろうさんのパソコンにある `index.html` をエディタ（メモ帳やVS Codeなど）で開きます。
4. コントロールキーとFキー（`Ctrl + F`）で検索を開き、`広告タグ貼り付け位置` と検索します。
5. 以下のような部分が見つかるはずです。

```html
                    {/* Google AdSense 等の第二広告プレースホルダー（記事下大枠用） */}
                    <div class="mt-12 text-center">
                        <!-- ■■■ 広告タグ貼り付け位置（ボトム） ■■■ -->
                    </div>
```

6. この `<!-- ■■■ 広告タグ貼り付け位置... -->` の1行を消して、代わりに**コピーした広告タグ（`<script>...`）をそのまま貼り付けます**。（トップの広告エリアも同様に行ってください）
7. ファイルを上書き保存します。
8. 変更した `public` フォルダを、再度Netlifyの管理画面（Deploysタブ）にドラッグ＆ドロップして上書きアップデートします。

これで、Webアプリ内に本物の広告が表示されるようになり、アクセス数と利用回数に比例して自動で広告収入が入るモデルの完成です！
